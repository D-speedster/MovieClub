# MovieClub

یه سایت فول‌استک برای دانلود فیلم و سریال که با React و Node.js ساختم.

## درباره پروژه

این پروژه یه پلتفرم کامل برای آپلود و دانلود فیلم و سریاله. کاربرا می‌تونن فیلم‌ها رو جستجو کنن، ببینن و دانلود کنن.

## فیچرها

- جستجو و فیلتر فیلم‌ها و سریال‌ها
- آپلود فایل (برای ادمین‌ها)
- دانلود فیلم و سریال
- سیستم ورود و ثبت‌نام
- طراحی ریسپانسیو
- سرعت بالا و UI ساده

## تکنولوژی‌ها

**Backend**
- Node.js
- Express.js
- Multer

**Frontend**
- React
- Axios
- Bootstrap

## ساختار پروژه
```
movieclub/
├── backend/
└── frontend/
```
